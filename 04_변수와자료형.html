변수 선언 방식

 
            var : 변수, 변수명 중복o(덮어쓰기), 함수레벨 scrope
            let : 변수, 변수명 중복 x, 블록레벨 scope
            const : 상수, 변수명 중복 x, 블록레벨 scope


            1순위 const*(JS 특정 요솔르 선택하는 경우가 많아 ㅇ변수에 고정시계등)


            1순위 : const
            2순위 : let
            3순위 : var

자바 스크립트 자료형

 
            자바 스크립트는 변수 선언 시별도의 자료형을 지정하지 않음
            -> 변수에 대입되는 값의


            -string(문자열)
            -number(숫자 -> 정수/실수모두 포함)
            boolean(논리값)
          object(객체) -> 배열(arry), 자바 스크립트 객체{k:v, k:v}
          function(함수)
          unfunction(정의되지 않은 변수) -> 변수가 선언만되고 값이 대입하지 않음

            (참고)
            null (참조하고 있는게 엇음) : null은 타입보다는 값(리터럴의) 의 개념
